# Frontend

React + TypeScript frontend for Vocab-Track.

## Prerequisite
Make sure you have Node.js installed:

- **Node.js version:** 20.13.1
- **NPM version:** 10.7.0 (comes with Node.js)

You can check your versions with:

```bash
node -v
npm -v
```

## Setup
```bash
cd frontend
npm install
npm run dev
```
